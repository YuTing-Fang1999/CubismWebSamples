### Editor
`vscode`  
download vscode: https://code.visualstudio.com/Download

### Environment
`node.js`  
download node.js: https://nodejs.org/en/


### How to Use
```
git clone https://github.com/Live2D/CubismWebSamples.git
```  
```
cd .\CubismSdkForWeb-4-r.4\Samples\TypeScript\Demo\
```  
```
npm install
```  
```
npm start
```
